# TaskManagerWPF — Менеджер задач (To-Do / Planner)

Простое и удобное приложение для управления задачами с возможностью установки приоритетов, категорий, фильтрации и сохранения данных в JSON. Разработано на WPF (.NET Framework 4.8) с использованием современного интерфейса.

## Функциональные возможности

- **CRUD задачи** — создание, просмотр, редактирование и удаление задач.
- **Категории и приоритеты** — задачи можно группировать по категориям (например, Работа, Личное, Учёба) и устанавливать приоритет (Low, Medium, High).
- **Фильтрация** — фильтрация списка по приоритету, категории и статусу выполнения.
- **Интерактивный статус** — отметка о выполнении ставится/снимается одним кликом по чекбоксу в списке задач.
- **Сохранение данных** — автоматическое сохранение задач в JSON-файл (`tasks.json`) при каждом изменении.
- **Современный интерфейс** — скруглённые углы, тени, цветные бейджи приоритетов, адаптивные элементы.

## Технологии

- **Язык:** C#
- **Платформа:** WPF (.NET Framework 4.8)
- **Сериализация:** Newtonsoft.Json
- **UI:** XAML, стили без использования сторонних библиотек

## Установка и запуск

1. **Клонируйте репозиторий** или скачайте исходный код.
2. Откройте решение `TaskManagerWPF.sln` в Visual Studio (рекомендуется версия 2019 или 2022).
3. Восстановите NuGet-пакеты (должен автоматически подтянуться Newtonsoft.Json).
4. Соберите решение (**Build → Rebuild Solution**).
5. Запустите приложение (F5).

## Использование

### Главное окно
- **Список задач** отображается в центре. Каждая задача показывает название, срок, категорию, приоритет (в цветном бейдже) и статус (галочка).
- **Фильтрация** — выберите приоритет, категорию или отметьте «Выполненные» и нажмите «Применить». «Сброс» возвращает полный список.
- **Добавить** — открывает окно создания новой задачи.
- **Изменить** — редактирует выбранную задачу.
- **Удалить** — удаляет задачу после подтверждения.
- **Отметка выполнения** — клик по чекбоксу в колонке «Статус» мгновенно меняет состояние и сохраняет его.

### Окно задачи
- Заполните название (обязательно), описание, срок, категорию, приоритет.
- Галка «Задача выполнена» устанавливает статус.
- Кнопка **Сохранить** — сохраняет изменения и закрывает окно.
- Кнопка **Отмена** — закрывает окно без сохранения.

## Дополнительные возможности

- Приоритеты отображаются цветными метками (зелёный — Low, оранжевый — Medium, красный — High).
- Просроченные задачи (срок раньше сегодняшнего дня) подсвечиваются красным цветом даты.
- Данные автоматически сохраняются в файл `tasks.json` в папке с исполняемым файлом.

## Примечания

- Для корректной работы убедитесь, что в системе установлен .NET Framework 4.8.
- При первом запуске создаётся файл с тестовыми категориями (Работа, Личное, Учёба). Список задач изначально пуст.
- В текущей версии управление категориями через интерфейс не реализовано (категории заданы программно). При желании можно расширить функционал.
